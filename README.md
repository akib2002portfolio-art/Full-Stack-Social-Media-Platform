# 🌐 Socializer – Full-Stack Social Media Platform

### 📌 Description

Socializer is a modern full-stack social media platform built with the MERN stack, designed to help users connect, share experiences, and engage with friends in real time. The platform provides essential social networking features including user profiles, posts, comments, likes, friend requests, and customizable account settings. Users can personalize their profiles, share images and thoughts, interact with content, and manage their social connections through a responsive and user-friendly interface. The application also includes secure JWT-based authentication, Cloudinary-powered image management, dark/light themes, and a mobile-first design approach for an optimal experience across all devices.

🚀 **Live Demo:** https://socializerme.vercel.app/

📖 **Documentation:** https://github.com/janaiscoding/socializer/tree/main/backend#readme

### ✨ Key Features

* User Authentication & Authorization (JWT)
* Customizable User Profiles
* Create, Edit & Delete Posts
* Image Upload & Management
* Like & Comment System
* Friend Requests & Social Connections
* Dark & Light Theme Support
* Responsive Mobile-First Design
* Cloudinary Image Optimization
* Secure Full-Stack Architecture

### 🛠️ Tech Stack

* React.js / Next.js
* Node.js
* Express.js
* MongoDB
* TypeScript
* Tailwind CSS
* Cloudinary
* JWT Authentication
